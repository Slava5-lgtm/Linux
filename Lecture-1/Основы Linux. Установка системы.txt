                                                                VirtualBox

Сначала скачиваем с сайта https://ubuntu.com/download/desktop/thank-you?version=22.04.2&architecture=amd64
и VirtualBox (виртуальноую машину) https://www.virtualbox.org/wiki/Downloads
Виртуализация в VirtualBox
● Межплатформенный продукт.
● Требуется поддержка аппаратной виртуализации в CPU.
● Широкие возможности по экспериментам.

Настройка виртуальной машины
● RAM от 2048 MB
● Диск от 20 GB
● Загрузка с установочного ISO

После того как скачал VirtualBox надо устоновит ту версию какая ОС на компьютери. После того как
устоновили на рабочем столе появиться ярлык (Oracle VM VirtualBox) запускаем его. И создаем Вертуальную
машиу и в ней установим нашу систему Linux: 

Создать -> Имя (на свое усмотрения) -> Тип (Linux) -> Версия (Ubuntu(64bit)) -> Основная память (не менее
2048 МБ но не заходить в красную зону) -> Создать новый вертуальный жеский диск (Размер диска не менее 20
ГБ) (поставить галочку) Выдилить место в полном размере -> Готово
Появляеться Вертуалная машина ее мы достраиваем чтобы устоновить Linux:
Настройки -> Система: Материнска плата (Порядок загрузки (снять галочку)Гибкий диск), Процесор (не
заходить в красную зону) -> Дисплей (не заходить в красную зону) -> Носители: Контролер: IDE (вставляем
скаченый файл (ubuntu.com)) -> Ауди отключаем -> Сеть: (поставить галочку) Включить сетевой адаптер, Тип
подключения NAT (он универсальный подходит для всех) -> OK -> Запустить -> Появляется  GMU GRUB version
2.06 Выбираем (Try or instull Ubuntu) ENTER -> Пявляеться экран Welcome, выбераем язык, Попробывать Ubuntu
-> Система загрузилась -> Идем в настройки (правый верхний угол) -> Параметры: Настройки экрана выбираем
нужное расширение (применить, разрешить), клавиатура + добовляем англиский язык (все настройки закрываем)
-> На рабочим столе есть ярлык Устоновить  Ubuntu 22.0402...-> Открываем: Русский язык (продолжить),
Раскладка клавиатуры (продолжить), Какие приложения вы хотите устоновить для начала: Минималые устоновки,
Другие загрузки:(поставить галочку) Загрузить обновления во время устоновки Ubuntu, Устоновить строеное
програмное обеспечение для графики и обновления Wi-Fi и дополнительных медиа форматов (На вертуалке не
реально) -> Устоновить сейчас -> Настраиваем часовой пояс -> Настройка имени, логина и.т.д -> После
устоноки надо перегрузить систему -> Извлечь диск (правый нижний угол(вторая иконка(IDE (правая кнопка
мыши (извлечь диск с привода)) иногда может отключиться автоматически))) -> Ввести пароль Enter -> Система
 загрузилась -> Пропустить (Нет не отпровлять информацию о системе) -> Далее-> Настраиваем

Работа в терминале. Нижний левый угол находим вкладку терминал. Левая сторона рабочего стола терминал
(правая кнопка мыши (Добавить в изброное)). Переходим в Terminal.md

                                                        Terminal

Cначало идем в терминал и смотрим какие пакеты есть командой sudo apt update. При команде sudo будет
требовать пороль вводим нажимаем Enter. Устоновим пакет mc будет спрашиват о согласии. После этого
устонавливаем sudo apt install openssh-server. После этого переходим в папку Network modes (сетевые
режимы)
