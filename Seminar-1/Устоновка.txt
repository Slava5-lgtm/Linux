                                    Установка виртуальной машины
С сайта (https://www.virtualbox.org/wiki/Downloads) качаем виртуальную машину и устонавливаем.
Далее нам нужен оброз диска нашей операционной системы которую мы будем учтанавливать. Переходим на сайт
(https://ubuntu.com/#download) -> Download -> Ubuntu Desktop -> Download 20.04.2 и качаем эту версию.
После скачивания переходим VirtualBox. Создать -> Имя (на свое усмотрения), Папка (на свое усмотрения),
оброз ISO (место куда качали ISO - файл отудо ег выбрать), поставить галочку (Пропустить автоматическую
установку) -> Далее (Основная память (не ставить болше половины), Процесоры (не ставить болше половины))
-> Далее (Создать виртуальный жеский диск (Размер диска 25ГБ) долше не где галочки не ставить) -> Далее
(Все провиряем если все устраивает продолжаем) -> Готово. Запускае машину кнопкой запустиь (зеленая
стрелка) -> Появляется окно (ubuntu (Работает)-Oracle VM VirtualBox (Выбираем (Try or install Ubuntu)
нажимаем Enter) Появляеться окно приветствия выбираем язык и устоновить Ubuntu, Раскладка клавиатуры ->
Продолжить -> Обновления и другие програмы (на свое усмотрения) -> Продолжить -> Тип устоновки (Стереть
диск и устиновить Ubuntu) -> Устоновит сейчас  -> Выбираем часовой пояс) -> Продолжить -> Кто вы (самое
главное имя пользователя и пороль) -> Продолжить -> Началась устоновка -> После устоновки нажать 
перегрузить.
Устонавливаем гостевые дополнения (приложения) - это такое приложения которое ставиться в вертуальную
машину и позволяет ей обшяться более активней с основной машиной. Выбираум устройство -> Подключить оброз
диска Дополнений гостевой ОС... (с лева появился диск (VBox_GAs_7.0.8)). Устонавливаем гостевые
дополнения нажимаем на диск (VBox_GAs_7.0.8). Открываеться папка VBox_GAs_7.0.8 в ней п.к.м выбираем
открыть в терминале попадаем сюда (диск (ubuntu@VirtualBoxGB:/media/ubuntuuVBox_GAs_7.0.8$)) вводим
команду
sudo apt update – обновить список пакетов
sudo apt install gcc make perl - установить пакеты gcc make perl
sudo ./VBoxLinuxAdditions.run - установка гостевых дополнени
sudo reboot - перегрезить систему
Устройство -> Обший буфер обмена -> Двухнаправленый (Можно будит копировать текстс Винды и встовлять в
Линукс )
Ctrl+Alt+T - открываеться терминал
Устройство -> Обшие папки -> Настройка обшей папки (открываються Обшие папки нажимаем на +(Добовленяет
новую обшую папку)) -> Путь к папке (это путь на моей хостовой машине (тоесть на Виндовс)) -> Точка
подключения (это место где она будет подключена в моей вертуальной машине) -> Авто-подлючение, Создать
постаяную папку (она будит автоматически постояно подключаться это будет постоян работаюшая папка) ставим
галочку, Только для чтения это если я не хочу чтобы из Линукса я мог что то менять. Менять мог бы только
из винды но так как галочка не стоит менять можно из винды и линукса -> ОК
Устонавливаем SSH сервер что бы мы могли к нему подключаться
sudo apt install openssh-server - Устонавливаем SSH сервер
В терминале настройки ->  параметры -> Основные -> Включить быструю клавишу доступа к меню
(убрать гадочу) ->  Безымянный: Звук - Подовать гудок (убрать гадочу), Курсор - форма i-образная
ip a - смотрим айпи адрес
		
				Подклюсения к интернету
Устройство -> Сеть -> Насстройка сети -> Адаптер -> Тип подключения -> Сетевой мост -> Intl(R) Ethernet Connetiom (2) 1219-V -> Винда -> Поиск -> cmd ->
ssh (имя пользователя)@Ip -a виртуальной машины -> Enter -> yes -> Enter -> пароль -> Enter
Ctrl+Z - свернуть процес
Ctrl+D - остоновить
jobs - посмотреть сколько свернутых команд (или скольеопроцесов в работе)
fg (цивра свернутого процеса) - открывает его
bd (цивра свернутого процеса) - будет работать в фоновом режиме
